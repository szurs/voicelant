 Fast and easy creation of applications

Voicelant - versatile, powerful and yet easy-to-development tool for quickly and easily create phone applications, IVR, with a low price. Visually intuitive graphical script editor VisualVT allows almost instantly recreate the on-screen picture of the script passing the call and then test it using simulator DisplayVT without connection to the working system. Voicelant opportunities can be expanded through plugins, that allows you to create complex distributed telephone systems. Voicelant will help you save time and money when implementing new services.
