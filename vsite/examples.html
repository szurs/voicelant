 Examples

Here are examples of scenarios and scripts for VBScipt Voicelant.

 Scenarios
Number Description Image Download
1 Scenario to select messages in different languages. multilang.pat
2 A scenario that allows you to listen to 5 times of the salary of their codes, introduced by way of tone. Information retrieved from the database on SQL - query. If you enter the "#", then the call is completed. worker.pat
3 Scenario to demonstrate the use of script VBScript (file Digits_ENG.vbs), which converts the numbers in sequence files. WAV for reading. saydigits.pat
4 Scenario selection message from the specified container services on time or by numbers. First, forward to 3 seconds to enter two DTMF digits, if the figures were introduced - plays a file named by the same numbers if the numbers were not introduced - plays a file with a name equal to the number of minutes each hour. timeswitch.pat
5 Example scenario recording a message in voice mailbox. Specifies the path to the sub-mailbox, plays the title, then checks have not exceeded your mailbox limit capacity in seconds, if not - write a message to the end of the box, if exceeded - warns of the overflow box and taken away in seconds. writetovmb.pat
6 Reading messages from a voice mailbox. Specifies the path to the sub-mailbox, plays the title, then in order read the messages from the mailbox. If you entered the number "4" - plays the previous message, if the "6" - next, if any other figure - repeats the current message. readvmb.pat
