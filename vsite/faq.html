 Frequently Asked Questions (FAQ)

 How to dictate the speech prompt?

In order to record voice prompt need a computer with an installed (or built) a sound card, microphone and headphones. To listen, edit and record voice (audio) files, you can use standard software included in the delivery of Windows - the Phonograph. This application allows you to record, play and edit sound files of format WAV. When recording it is desirable to set the sound quality: The format - CCITT A-Law (u-law), Properties - 8000 Hz, 8 bit, Mono, 8 KB / sec. You can also use multimedia - applications other suppliers that offer a wide range of opportunities to work with audio files:

  • CollEdit 96, CoolEdit Pro, CoolEdit 2000, Adobe Audition (www.adobe.com/products/audition/main.html) - a set of professional editors audio files of Adobe.
  • Sound Forge Audio Studio 8 (www.sonymediasoftware.com/) from Sony - one of the leading sound editors. It has powerful editing functions, allows to build plug-ins, has a comfortable modern interface.
  • WaveLab 5 (www.steinberg.de) of Steinberg Media Technologies AG - includes into group of leading sound editors. It has powerful tools for editing and sound processing, allows to build plug-ins, supports many formats of audio files.

 It supports fax?

Support for fax machines is achieved in two ways:

  1. For systems that are equipped with voice cards with fax resources - a challenge of the program functions;
  2. For systems which are based on voice boards without fax resources through installing additional fax board, on which programmatically will be transfer a fax data (a fax-board should be connected to other boards by SC-bus or CT-bus);

 Which databases are supported Voicelant?

Supports all databases that have OLEDB provider or ODBC driver to access them - ORACLE, MS SQL Server, MySQL, Interbase (Firebird, Yaffil), MS Access, IBM DB2, PostgreSQL, Sybase, Informix, FoxPro, etc. In order to accelerate access and empowerment can be used the specialized API which connected by plug-ins.

 How can I send SMS?

There are different ways for sending SMS:

  • Sending SMS to email via gateway "e-mail - SMS", which is available to each communication operator.
  • For simple systems - sending SMS via the GSM-phone, connected to a computer using the appropriate plugin.
  • For corporate systems - sending SMS by means of SMPP-protocol directly into SMS-center.

 It supports Text-To-Speech (TTS)?

Now support Text-To-Speech is implemented through plugin TTS, which is working on SAPI 5 engine and converts the text into a sound file with a name that is passed to play. Then this file is deleted.
