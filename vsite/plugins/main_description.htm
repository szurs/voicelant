Описание дополнительного ПО

Данная справка предназначна для описания дополнительного программного обеспечения для Voicelant. Это программное обеспечение подразделяется на категории:

·Плагины Voicelant, раширяющие возможности базового набора программного обеспечения Voicelant.
·Комплекты сценариев Voicelant, скриптов базы данных и вспомогательного ПО, составляющих законченные приложения для различных областей применения.
