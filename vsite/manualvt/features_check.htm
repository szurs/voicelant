Scenarios created with the help of VisualVT, are testing with the help of programm DisplayVT, which can be run from the shell VisualVT, or separately.

 

When verification is done, run the script, closest to the actual call when establishing a connection.

 

Before you start connecting all the plugins running, you can enter additional data call. Speech files are played via a sound card, the user may enter DTMF signals using a digital keyboard.

 

Thus, you can customize the work of service without connecting them to the working system.
