Редактировать (Edit)

 Top  Previous  Next

Команда <Edit> предназначена для редактирования простейших строк (например, номеров телефонов) и выполения арифметических операций. Для редактирования сложных текстов нужно применять другие средства, например .

 

 

Функции для работы со строками:

·<Concat> - конкатенирует (складывает) две строки, полученные из стандартных источников и помещает результат в заданную переменную. (на примере: складывает строку "8800" и строку из переменной DNIS, результат - в переменной myvar)
·<SubStr> - Удаляет из заданной строки <Value 1> подстроку <Value 2><Result>.
·<Mid> - Извлекает из заданной строки <Value> подстроку длиной <Count> символов, начиная с индекса <Index>, помещая результат в заданную переменную <Result>. Если <Count> не задано, то до конца исходной строки. <Index> начинаеся с 0. 
·<Right> - извлекает из <Value> последние <Count> символов и помещает их в <Result>.
·<Left> - извлекает из <Value> первые <Count> символов и помещает их в <Result>
·<TrimRight> - удаляет справа <Count> символов в строке <Value>.
·<TrimLeft>- удаляет слева <Count> символов в строке <Value>.
·<Length> - помещает в <Result> длину строки <Value>.
·<InStr> - проверяет, содержится ли в заданной строке <Value 1> подстрока <Value 2>. Условия выполнения - если содержится - , если нет - .

 

Функции для выполнения арифметических операций:

·<Add> - складывает арифметически значения данных <Value 1> и <Value 2>, полученных из стандартных источников. Результат сохраняет в <Result> (м.б. переменной или счетчиком цикла)
·<Sub> - вычитает из <Value 1> значение <Value 2>, Результат - в <Result>
·<Mul> - перемножает значения <Value 1> и <Value 2>. Результат - в <Result>.
·<Div> - помещает в <Result> частное от деления <Value 1> на <Value 2>.(на 0 делить нельзя).
·<Mod> - помещает в остаток от деления <Value 1> на <Value 2>.(на 0 делить нельзя).
