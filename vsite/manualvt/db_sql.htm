Закладка <Query source> предназначена для задания SQL - запроса.

Строка SQL - запроса может быть получена из стандартных источников:

 

 

·<File> - ввести или выбрать кнопкой имя файла, в котором находится SQL - запрос;
·<Text>ввести SQL - запрос (кнопка - большой текст);
·<Variable> - ввести или выбрать переменную с SQL - запросом;
·<Record> - набрать поле записи, содержащее необходимый SQL - запрос.
