Произнести номер телефона

 Top  Previous  Next

<Phone number> - читает в линию число, как номер телефона;

Цифровая строка с номером телефона может быть получена из стандартных источников:

 

 

·<File> - ввод или выбор кнопкой файла, в котором находится числовая строка с номером телефона;
·<Text>
·<Variable> - ввод или выбор имени переменной, содержащей числовую строку с номером телефона;
·<Record> - набор номера поля записи, в котором находится строка с номером телефона;

 

Цифровые номера телефонов  (без разделителей) произносятся так:

·1 -значные ("5" - "пять")
·2 - значные ("23" - "двадцать три");
·3 - значные ("457" - "четыреста пятьдесят семь");
·4 - значные ("4567" - "сорок пять шестьдесят семь");
·5 - значные ("85327" - "восемь пятьдесят три двадцать семь")
·6 - значные ("853271" - "восемьдесят пять тридцать два семьдесят один")
·7 - значные ("8532716" - "восемьсот пятьдесят три  двадцать семь шестнадцать")
·8 - значные ("85327169" - "восемь пятьсот тридцать два семьдесят один шестьдесят девять");
·9 - значные ("853271690" - "восемь пять триста  двадцать семь шестнадцать девяносто");
·10 - значные ("8532716904" - "восемьсот пятьдесят три двести семьдесят один шестьдесят девять ноль четыре");
·11 - значные ("85327169041" - "восемь пятьсот тридцать  два семьсот шестнадцать девяносто сорок один");

 

Если в номерах телефонов есть разделители (-,.:), то телефоны читаются по разделителям, согласно значности (например: "8-902-88-54-190" - читается "восемь девятьсот два восемьдесят восемь пятьдесят четыре сто девяносто").

 

Звуковые файлы, из которых составляется фраза номера телефона находятся в подкаталоге .\DUTY.
