Запрос к базе данных (Database)

 Top  Previous  Next

Команда <Database> позволяет выполнить SQL - запрос к базе данных.

 

Доступ к различным базам данных (ORACLE, MS SQL Server, MySQL, Interbase, MS Access, IBM DB2, PostgreSQL, Sybase, Informix, DBF) может осуществляться разными способами (OLE DB, ODBC, специализированные API) посредством установленных плагинов Voicelant.

 

В стандартный комплект поставки Voicelant входит плагин MSSQL.pvt - для доступа к базе данных MS SQL Server через OLE DB провайдер. Плагин ODBC.pvt - для доступа к БД через ODBC драйвер, можно заказать на сайте Voicelant.

 

В зависимости от запроса данные могут не возвращаться (SQL UPDATE или INSERT), возвращаться в виде записи Record (SQL - SELECT) или в переменных (для хранимых процедур).

 

 

·<DB service> - закладка для выбора сервиса плагина, который обеспечивает доступ к базе данных;
·<Parameters> - закладка для задания параметров, передаваемых в SQL - запрос (если необходимо);
·<Query source> - закладка для задания SQL - запроса;
·<Connection> - дополнительная строка соединения;
·<Wait 8 sec> или <Not Wait> - ждать или не ждать окончания выполнения SQL - запроса;
·<Run query (wait 8 sec) from text "select * from tb" in database "MS-SQL"> - строка комментария (изменяется);
