Выбор по ключу (Switch)

 Top  Previous  Next

Команда <Switch> производит сравнение данных ключа (полученных из стандартных источников) с данными вариантов выбора и выполняет сценарий, соответствующий варианту совпадения данных.

 

·<Cases> - закладка задания количества вариантов;
·<Number of cases> - набор количества вариантов выбора;
·<Source> - закладка  для задания данных ключа (см. стандартные источники данных):

 

Пример команды <Switch>, в которой в качестве ключа выбрана переменная DTMF и задано 3 варианта выбора:

 

Если переменная DTMF равна 1, то читается файл "1.wav", если равна 2  - то файл "2.wav", если не 1 и не 2, а любому другому символу - то "not.wav".

Источники данных для вариантов выбора можно выбирать через правую кнопку мыши -> <Properties>.

 

При сравнении можно использовать "подстановочные" символы (wildcard) - "_" и "%", где "_" - заменяет любой один символ, а "%" - может заменять любое количество символов. Например, строки "My name is Voicelant" и "M_ na_e % Voicelant" равны.

 

Количество вариантов выбора для команды <Switch> можно добавлять через <Properties> -> <Number of cases>. Удалять варианты выбора  можно через правую кнопку мыши -> <Delete>.
