Установка переменных

 Top  Previous  Next

Закладка <Variables> предназначена для установки переменных, передаваемых в сценарий задания, который будет выполняться выбранным каналом.

 

STR или LONG. Направление передачи - всегда INOUT, т.к. текущий сценарий своего канала передает перменные, а канал-получатель задания принимает эти переменные и может использовать их в своем сценарии.

 

 

·<Insert> - кнопка для добавления переменных (вставка после выбранной);
·<Delete> - кнопка для удаления выбранной переменной;
·<DeleteAll> - кнопка для удаления всех переменных;
·<№> - номер переменной по порядку;
·<Type> - тип переменной: STR - строка, LONG - число;
·<In/Out> - направление передачи переменной: - в обе стороны;
·<Source> - стандартный источник для переменной:
