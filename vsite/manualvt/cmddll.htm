Команда <DLL> DLL.

 

Поддержка DLL реализована с помощью плагина DLL.pvt, который входит в стандартный комплект поставки Voicelant.

 

Сигнатуры для используемых функций:

       int __stdcall FFuncZero();

       int __stdcall FFuncOne(void*);

       int __stdcall FFuncTwo(void*,void*);

       int __stdcall FFuncThree(void*,void*,void*);

 

В вызываемых функциях можно использовать до 20 параметров. Тип параметров функции char* (STR) или int* (LONG), длина передаваемой и принимаемой строки не должна превышать 255 символов;

 

Параметры могут быть входящими (IN), исходящими (OUT) или передаваться в обе стороны (INOUT). Из всех стандартных источников данных только переменные могут быть исходящими или двусторонними. Возврашаемое функциями значение  - параметр с порядковым номером 0,  всегда существует, даже если нет других переменных для передачи. После выполнения функции DLL ее значение можно получить из стандартных источников: через поле 0 записи <Record> или переменную "retcode".

 

 

·<Name DLL> - ввести имя или выбрать кнопкой нужную DLL;
·<Funtion> - ввести имя вызываемой функции;
·<Insert> - кнопка для добавления параметров (вставка после выбранного);
·<Delete> - кнопка для удаления выбранного параметра;
·<DeleteAll> - кнопка для удаления всех параметров;
·<№> - номер параметра;
·<Type> - тип параметра: STR - строка, LONG - число (для 0 параметра только BOOL);
·<In/Out> - направление передачи параметра: IN - входящий параметр, OUT - исходящий параметр, INOUT - в обе стороны ;
·<Source>- стандартный источник для параметра:

 

 

·<Exec function "GetNumber"> from dll "MyDLL"> - строка комментария (изменяется);
