При разработке сценария Voicelant с помощью визуальной среды VisualVT можно использовать следующие команды:

 

· - воспроизводит сообщения в телефонную линию различными способами.
· - записывает из телефонной линии файлы сообщений.
· - принимает из телефонной линии символы (цифры) тонального набора (DTMF).
· - набирает заданный номер и устанавливает соединение.
· - удаляет сообщения в контейнере услуги.
· - устанавливает фигурные скобки или задает цикл для данной группы команд сценария.
· - завершает выполнение текущего цикла или всего сценария с завершеним вызова.
· - создает новую переменную либо перезаписывает старую.
· - устанавливает некоторые "постоянные" значения параметров вызова.
· - подключает сценарий из указанного файла.
· - предназначена для редактирования простейших строк (например, номеров телефонов) и выполения арифметических операций.
· - производит сравнение данных.
· - выбирает по ключу ветвь для выполнения.
· - позволяет выполнить SQL - запрос к базе данных.
· - выполняет скрипт, написанный на языке VBScript.
· - вызывает функцию из заданной библиотеки DLL
· - позволяет выполнить запрос к заданному внешнему модулю - плагину.
· - производит поиск канала по заданным критериям и сохраняет ссылку на него.
· - посылает на выполнение выбранному каналу новое задание в виде сценария.
· - связывает два голосовых канала, свой и удаленный,
