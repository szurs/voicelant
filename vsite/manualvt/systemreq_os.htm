Requirements for the operating system

 Top  Previous  Next

Voicelant supports the following operating systems:

 

Windows NT (Workstation и Server);

Windows 2000 (Professional и Server);

Windows 2003 Server;

Windows XP;

 

 

All operating systems should be installed the latest version of ServicePak's.

 

Also desirable to install Windows Update's (http://www.microsoft.com/downloads/search.aspx?displaylang=en&categoryid=7)
