Tab <Inbound> - setting parameters for incoming calls.

·<Pattern (scenario) for incoming calls> - the path to the file the initial script - this script is loaded first when an incoming call (usually the script files are stored in a subdirectory \PATTERN\);
· <For analog lines only. Map of conformity of channels to the called number:> - only for analog lines. Map matching channel number and phone number lines, to which the channel is connected (to fill the variable DNIS when you call).
