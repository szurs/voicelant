Задание параметров

Закладка <Parameters> предназначена для передачи параметров в SQL - запрос.

Например, при выполнении запроса "select * from tb where Num= ? and Status = ?" символы "?" заменяются на данные параметров по порядку, начиная от первого. Параметр с номером 0 используется для возвращаемого значения выполнения хранимых процедур (например, "? = call stored_proc ?,?", где первый знак вопроса - возвращаемое значение).

 

 

·<Insert> - кнопка для добавления параметров (вставка после выбранного);
·<Delete> - кнопка для удаления выбранного параметра;
·<DeleteAll> - кнопка для удаления всех параметров;
·<№> - номер параметра;
·<Type> - тип параметра: STR - строка, LONG - число (для 0 параметра только );
·<In/Out> - направление передачи параметра: IN - входящий параметр, OUT - исходящий параметр, INOUT - в обе стороны (для 0 параметра только RET - возвращаемый);
