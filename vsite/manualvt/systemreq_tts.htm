Speech Synthesis (Text To Speech)

 Top  Previous  Next

Voicelant supported Speech Synthesis through plug-in TTS, which you can ordering from site www.voicelant.com.

 

Prior to the use of speech synthesis capabilities need to install Microsoft SAPI 5, which can be downloaded from http://download.microsoft.com/download/speechSDK/SDK/5.1/WXP/EN-US/speechsdk51.exe

 

Also need to install TTS engines, which are supplied by different manufacturers. For example:

 

·http://www.digalo.com/
·Elan Systems (http://www.elanspeech.com/) - Elan SaySo 4.6 Evaluator
·Lernout&Haspie - L&H TruVoice& TTS3000 Voice Engines
·Microsoft - Microsoft Speech Engine
·ScanSoft (http://www.scansoft.com/) - ScanSoft RealSpeak
·Cepstral (http://www.cepstral.com/) - Cepstral SwiftTalker 3.0

 

They have different number of supported languages, vocable speech quality and price..  .
