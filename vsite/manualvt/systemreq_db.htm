Access to databases

 Top  Previous  Next

The command of Voicelant through the appropriate plug-ins allow access to different databases.

 

Plugin MSSQL.pvt (data source - LINK) is working by default with the database MS SQL Server (or MSDE) via OLE DB (through the Data Link File mssql.udl). Also, this plugin is suitable for use with other sources and databases - ODBC, MS Access, Oracle, DB2, Interbase, MySQL and others when setting up a connection to the file mssql.udl.

The computer must have the appropriate OLE DB Providers.

 

New plug-ins interact with databases through other interfaces (ODBC, branded) you can download from the site www.Voicelant.com.

 

When you use ODBC the appropriate drivers must be installed.
