Выбор сервиса для доступа к базе данных

 Top  Previous  Next

Закладка <DB service> позволяет выбрать имя сервиса из предлагаемого списка для плагина, предназначенного для доступа к необходимой базе данных.

 

Например, плагин MSSQL.pvt, входящий в комплект поставки Voicelant, при запуске программы Voicelant подключается к базе данных MS SQL Server посредством OLE DB провайдера, данные для соединения берутся из файла MSSQL.udl. Этот плагин имеет два сервиса (алиаса) - "MSSQL"и "MS-SQL", которые могут использоваться для различных целей (например "MSSQL"для доступа к локальным таблицам базы, а "MS-SQL" к внешним серверам БД - linked server).

 

Плагин ODBC.pvt позволяет получить доступ к данным посредством ODBC - драйвера, для подключения  необходимо ввести строку соединения <Connection> (например, "DSN=TestDSN;UID=;PWD=;").

 

·<Plugin> - существующий плагин;
·<Service Name> - имя сервиса для выбранного плагина (м.б. несколько на один плагин);
·<Connection> - дополнительная строка соединения;
