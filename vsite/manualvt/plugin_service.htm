<Plugin service> - закладка для выбора сервиса нужного плагина.

 

Все плагины установленные в  отображаются списком со своими сервисами. Некоторые плагины могут иметь несколько сервисов (например, для разделения нагрузки). Такие плагины как DLL.pvt, MSSQL.pvt, Script.pvt используются встроенными командами Voicelant - , , . Плагин Service.pvt используется для внутренних целей Voicelant. Остальные плагины устанавливаются по мере необходимости. 

 

 

·<Plugin> - путь до файла плагина;
·<Service Name> - имя сервиса для выбранного плагина (м.б. несколько на один плагин);
·<Connection> - дополнительная строка соединения;
