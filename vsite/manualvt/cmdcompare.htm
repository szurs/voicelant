Сравнить данные (Compare)

 Top  Previous  Next

Команда <Compare> производит сравнение данных, полученных из стандартных источников:

 

·<Parameter 1> - закладка для получения данных первого сравниваемого параметра;
·<Parameter 2> - закладка для получения данных второго сравниваемого параметра;
·<Compare sign> - знак сравнения (= - равенство,> - больше, < - меньше):

 

·< = Equal (may be used wildcard: _ and %)> - проверка на равенство;
·< > Greater> - проверка на "больше";
·< < Less >- проверка на "меньше";

 

Если первый и второй параметр - чисто цифровые данные, то эти числа сравниваются арифметически. Если параметры строковые, т.е. содержат хотя бы один нецифровой символ, то сравнение строк производится лексикографически (регистр не учитывается).

 

При сравнении строк можно использовать "подстановочные" символы (wildcard) - "_" и "%", где "_" - заменяет любой один символ, а "%" - может заменять любое количество символов. Например, строки "My name is Voicelant"и "M_ na_e % Voicelant" равны.
