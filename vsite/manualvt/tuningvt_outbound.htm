Tab <Outbound> - setting parameters for outgoing calls.

·<Pattern (scenario) for outgoing calls> - the path to the file of scenario for outgoing calls - this script is loaded after a timeout (see below) (usually the script files are stored in a subdirectory \PATTERN);
·<Timeout between requests on tasks of outgoing calls > - time in seconds between requests to get task for autodialing. Each channel that can handle outbound calls from time to time loads the script (see above), which checks tasks for autodialing and if the task is exist - executes a call from the scenario).
