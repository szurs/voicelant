Цикл (Loop)

 Top  Previous  Next

Команда <Loop> устанавливает фигурные скобки, которые группируют команды сценария. Также для данной группы команд сценария можно задать циклическое выполнение, где счетчик цикла может быть получен из стандартных источников.

 

 

·<Braces>устанавливает фигурные скобки.
·<Loop> - устанавливает цикл, количество повторений можно получить из стандартых источников:

 

 

·<File> - ввести или выбрать кнопкой имя файла, в котором находится число повторений цикла;
·<Text> - ввести число повторений цикла (например, 5 раз);
·<Variable> - ввести или выбрать переменную с числом повторений цикла;
·<Record> - набрать номер поля записи, содержащее необходимое число повторений цикла;

 

Скобки и циклы легко можно преобразовать друг в друга (т.к. цикл с числом повторений = 1 - это просто скобки).

 

Также группы команд сценария, ограниченные скобками (или циклом) можно записать в отдельный файл (правой кнопкой мыши - Save to file + Include), который затем можно подключать с помощью команды .

 

Цикл завершается после выполнения заданного числа раз или, если нужно срочно выйти из цикла - командой - <Break current loop>.
