Work by the scenario

 Top  Previous  Next

All work on the processing of calls is carried out on the scenario, written in a script-like language, which has the following features:

 

· a small number of commands;
· the possibility of branching and cycles;
· using variables;
· reduction of the types of data - only string or numerical;
· orthogonal syntax of scenario oriented to subject area;
· using external modules for access to remote resources (databases, e-mail, Internet, other software blocks);
· the possibility to make speech phrases by compiled synthesis method that allows you to pronounce a numerical data, such as debt, the phone numbers, date, time, etc.;

 

The tool for easy creation of scripts is a visual graphical editor VisualVT.
