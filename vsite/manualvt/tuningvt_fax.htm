Tab <Fax> - tuning possibilities faxing.

· <Use Fax> - if checked, the system Voicelant supports faxing and then choose the type of fax resources (see below);
· <Use fax resource on board> - if selected, system is used voice cards with on-board fax resources;
·<Use GammaLink fax> - if selected, system is used special fax cards (CP/6SC etc.).
