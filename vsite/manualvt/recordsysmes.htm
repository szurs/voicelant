<Directory for system messages> - звуковой файл записывается в подкаталог для системных сообщений Voicelant .\SERV_MES. Системные сообщения Voicelant - это библиотека звуковых файлов, которая содержит общие подсказки (announcements) Voicelant. Имя файла для записи может быть получено из стандартных источников:

 

 

· <File> - ввести или выбрать кнопкой имя файла (по умолчанию - расширение .wav). Если файл находится в подкаталоге .\SERV_MES (стандартный подкаталог для системных звуковых файлов Voicelant), то путь до файла указывать не обязательно;
· <Variable> - ввести или выбрать имя переменной, в которой находится имя файла;
· <Record> - набор номера поля записи, в котором находится имя файла;
