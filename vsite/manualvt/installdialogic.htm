Installing the Dialogic boards and software

 Top  Previous  Next

Before you install the software insert the card(s) Dialogic into a free slot on the PC (ISA или PCI) (see documentation).

 

For Voicelant needs to install on a computer software Dialogic SR5.1.1 or higher.

 

Service Release 5.1.1 and Service Pack 1 for it is freely available on the website http://www.dialogic.com and are suitable for most boards Dialogic. Featured Boards require the installation of Feature Pack 1.

Most newer boards require the installation of Service Release 6.0 + Feature Pack 1 for SR6.0.

 

The installation process on example SR5.1.1 (typical passage for analog cards):

·Run SETUP from the supplies at Intel Dialogic System Software System Release 5.1.1 for Windows.
·Follow on sections with a choice of options to default.
·Restart the computer.
·Install Service Pack 1 for SR5.1.1.
·Restart the computer.
·Use the Dialogic Configuration Manager - DCM configure parameters of the card.
·Start the service Dialogic.

 

If you use configuration Voicelant with digital devices, or need to configure specific settings in the installation process may need to select additional settings. In all cases, see the original documentation Dialogic, which is supplied in the kit.
