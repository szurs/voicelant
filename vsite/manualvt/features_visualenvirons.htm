Visual environment for the creation of services

 Top  Previous  Next

To quickly and easily create all variety of services serves a visual graphical editor VisualVT, which has the following features:

 

· intuitive interface;
· multi-window environment;
· graphical presentation of commands;
· tree-like structure of scenario;
· cut, copy, paste;
· developed system of help;
· possible to verify the written scenario without connecting to the working system (DisplayVT);
