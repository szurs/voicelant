For quick and easy creation of all variety of services you can use visual graphical tool - VisualVT:

 

 

Features VisualVT:

· multi-windows interface that allows you to dynamically switch between the fragments of the scenario;
· graphical representation of commands;
· tree-like structure of the scenario, which allows you to develop scripts, saturated many commands;
· cut, copy, paste - for ease of creating scenarios;
· self-commented submission of scenario;
· run the program DisplayVT if you want to test the script;
· run the program TuningVT if you want to set parameters Voicelant;
· developed system of help;
· output script to print;
· tooltips;
