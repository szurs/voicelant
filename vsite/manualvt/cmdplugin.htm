.

Команда <Plugin>позволяет выполнить запрос к заданному внешнему модулю - плагину.

 

Плагины - это программные модули, расширяющие возможности Voicelant. Они используются для передачи, получения, обработки и преобразования данных.

 

Набор плагинов для системы зависит от конкретных задач. Дополнительные плагины можно подключать по мере небходимости. Основные плагины входят в поставку Voicelant, другие поставляются по желанию заказчика.

 

 

·<Plugin service> - закладка для выбора сервиса нужного плагина;
·<Parameters> - закладка для задания параметров, передаваемых в плагин (если необходимо);
·<Command source> - закладка для задания команды для плагина;
·<Connection> - дополнительная строка для задания параметров соединения с плагином;
·<Wait 20 sec> или <Not Wait> - ждать или не ждать окончания выполнения работы плагина;
·<Run service "TTS5" from file "plugtts.txt"> - строка комментария (изменяется);
