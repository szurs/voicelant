Tuning Voicelant - TuningVT

 Top  Previous  Next

The program TuningVT is designed to settings Voicelant, which are stored in a file parmvt.ini.

 

Plugin settings are stored in files <name plugin>.ini in subdirectory \PLUGINS.
The program TuningVT can be run as a standalone application or from the program ServerVT, selecting the appropriate menu item <Options> - <Set Parameters>.

 

All parameters Voicelant are divided into groups that can be configured by selecting the tab of program TuningVT:

 

· <System> - setting the system parameters;
· <Inbound> - setting the parameters of incoming calls;
· <Outbound> - setting options for outgoing calls;
· <Start> - parameters checked at startup Voicelant;
· <Protocol> - setting protocols;
· <Alarm> - notification of material events Voicelant;
· <Log> - choose the type of tracking and maintenance of log-files;
· <Fax> - setting faxes;
· <Plugins> - setting plugins;
