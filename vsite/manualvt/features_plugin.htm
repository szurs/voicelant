Voicelant feature is the ability to connect additional functionality through the use of software modules written by third-party vendors - the plug-ins. To write such modules Voicelant published programming interface. These commands Voicelant, as , , implemented through plug-ins included in the package.

 

Thus Voicelant functions can be divided into the phone part, which provides processing calls, it displays data in different formats, and a part which performs data processing using plug-ins.

 

Examples of plug-ins for the extension - different ways to access databases (OLE DB, ODBC, branded interfaces), HTTP, FTP, Telnet access to the data, sending and receiving e-mail, SMS, proprietary interfaces, non-standard data sources (rare OBD system Industrial automation - sensors). When fully developed system of plugins Voicelant can not even use the phone part of their work, and deal only with the processing and dispatch of data.
