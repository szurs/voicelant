Установить параметры (Set)

 Top  Previous  Next

Команда устанавливает некоторые "постоянные" значения параметров, необходимых для текущего вызова. Обычно вызывается в начале обработки сценария.

 

·<Work Directory(Box)> - установка рабочего подкаталога контейнера услуги;
·<Capacity of Box (minutes)> - задание максимальной емкости контейнера услуги (в минутах, по умолчанию - 2 минуты);
·<Duration of message (seconds)> - задание максимальной длительности записываемого сообщения;
·<Attribute of message> - чтение атрибутов сообщения;
·<Detect DTMF (ALL or terminator)> - на какие сигналы DTMF реагировать на все - по умолчанию или terminator - любой из символов DTMF);
·<Condition (TRUE or FALSE)> - задание условия;

 

Значения параметров можно получить из стандартных источников:

 

·<File> - ввести или выбрать кнопкой файл, в котором находится значение параметра;
·<Text> - ввести значение параметра;
·<Variable> - ввести или выбрать имя переменной, содержащей значение параметра;
·<Record> - набор номера поля записи, в котором находится значение параметра;
