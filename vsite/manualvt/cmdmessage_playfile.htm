Выбранный файл

 Top  Previous  Next

<Play file> - воспроизводит выбранный файл в линию (в зависимости от типа файла: звуковой - проигрывает, факсимильный - пытается передать факс);

Ссылку на файл можно получить из стандартных источников:

 

 

·<File>ввести или выбрать кнопкой имя файла (по умолчанию - расширение .wav). Если файл находится в подкаталоге .\SERV_MES (стандартный подкаталог для системных звуковых файлов Voicelant), то путь до файла указывать не обязательно;
·<Variable> - ввести или выбрать имя переменной, в которой находится имя файла;
·<Record> - набор номера поля записи, в котором находится имя файла (или массив (array) файлов);

 

Если файл для воспроизведения не найден, то выполнение сценария продолжается.
