Для разных языков командой можно подключать различный файлы сценариев, где информация диктуется на выбранном языке. Например:

 

 

В данном сценарии при нажатии клавиши "1" телефонного аппарата подключится сценарий из файла "English.pat",  при нажатии клавиши "2" - файл "Spanish.pat", при нажатии клавиши "3" - файл "French.pat", а при нажатии клавиши "0" - прозвучит помощь.

 

В разных странах правила чтения числовых значений, таких как денежные величины, номера телефонов, дата, время, могут различаться. Для возможности написания скрипта VBScript, который программирует эти правила, в обработчик скрипта (команда ) vtarray" (из сценария не видна), значения которой в виде массива можно извлечь из записи <Record>. Например, в теле скрипта можно написать:

vtarray(0)="c:\Voicelant\serv_mes\ONE.wav"

vtarray(0)="c:\Voicelant\serv_mes\TWO.wav"

vtarray(0)="c:\Voicelant\serv_mes\THREE.wav"

vtarray(0)="c:\Voicelant\serv_mes\FOUR.wav"

и затем прочитать запись <Record>, как массив (array) файлов с помощью команды .
