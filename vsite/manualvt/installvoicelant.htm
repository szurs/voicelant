Installing Voicelant

 Top  Previous  Next

Just run SETUP.

 

As the installation program will ask for additional parameters that can be left by default.

On selected drive will create a directory \Voicelant to accommodate all the necessary files.

The main program files:

 

·VisualVT.exe - graphical interface to quickly and easily create scenarios, presents scenarios in visual form;
·DisplayVT.exe - program run scenarios can check a scenario before the installation of working applications;
·TuningVT.exe - setting Voicelant, you can set the basic parameters are also built into the main application;
·ServerVT.exe - the main application, to serve the incoming and outgoing calls on a given scenario;
