 Features

 Key Features

easy installation and configuration using the graphical tools;

flexible and easy-to-use tool to quickly create all variety of services - a visual scenario editor VisualVT;

comprehensive system of commands for creating scenarios focused on a subject area;

scenarios in a tree structure that allows you to create applications with a lot of commands without a loss of visual inspection;

unique logic service container with the specialized commands to locate these services, brings together many services such as voice mailboxes, auto, etc. in one application;

creating a test call via the sound card, without connecting to the working system - a simulator DisplayVT;

connection plugins that enhance Voicelant and to complete a specific system, depending on need;

close integration with various data sources: files (text, DBF, etc.), databases (MS SQLServer, Oracle, MySQL, Interbase, DBF and others through interfaces OLE DB, ODBC and proprietary), Internet (HTTP, FTP, SMTP , Telnet), non-standard data sources (sensors, control systems, etc.) through the plug-plug;

possibility to execute scripts VBScript, run third party programs that perform the functions of the software library DLL using the embedded plug-ins;

integration with the Internet (WEB, FTP, E-mail), allows to build distributed systems with the management services via WEB - interface;

the possibility of creating systems that use mobile services: SMS, MMS, USSD, WAP, GPRS, etc.;

scalability: The number of telephone channels serviced Voicelant - from 2 up to 480 per server;

use to connect to the telephone network of the entire diversity of interfaces: analog telephone lines, digital channels E1 (ISDN PRI, R2, SS7);

 VisualVT

Work Voicelant determined scenario, implementing the desired functionality when you receive incoming calls or initiate outgoing connections. The scenario is a program written in a simple script-like language. Create a scenario, you can quickly and easily using a visual graphical editor VisualVT, which has the following features:

  • small number of commands and data types;
  • the possibility of branching and cycles;
  • use variables;
  • orthogonal syntax oriented on subject area;
  • maintaining the local fragments of script files and connect them as needed;
  • use of external modules (plugins) to access remote resources (databases, e-mail, Internet, other software blocks);
  • multiwindows interface that allows you to dynamically switch between the fragments of the script;
  • graphical presentation of commands;
  • tree-like script, which allows to develop scenarios that saturated the large number of commands;
  • cut, copy, paste - to ease the creation of scenarios;
  • self-commented presentation scenario;
  • opportunity to run the program simulator DisplayVT to test the script;
  • opportunity to run the program TuningVT to set Voicelant;
  • developed system of help;
  • print of scenario;
  • tooltips;

 DisplayVT

Verification of the scenario created by VisualVT, produced by program DisplayVT. Program DisplayVT emulates the telephone device through a sound card and allows to test and debug the scenario in the conditions as much as possible approached to actual call flow. Thus it is possible to adjust work of services without their connection to working system.

  • opportunity to run as from environment VisualVT, and separately;
  • ability to load a script file(if you start DisplayVT as a separate application);
  • visual display of callflow the scenario;
  • connection to all installed plugins;
  • enter additional data - dialed (called) number and number of the calling subscriber;
  • playing greeting and other messages through a sound card;
  • enter DTMF tones using the numeric keypad;

 Plugins

A feature Voicelant is the ability of connect plugins - software modules that enhance the system, which is used for transmitting, receiving, processing and data conversion. A set of plugins for a particular system depends on the desired functionality. Additional plugins can be connected as needed. Basic plug-ins included in the delivery of Voicelant, others can be purchased on site Voicelant. Custom plugins can be developed on request.
Such embeded commands as , , implemented through plug-ins included in the package. Examples of plug-ins for the extension - different ways to access databases (OLE DB, ODBC, branded interfaces), HTTP, FTP, Telnet access to the data, sending and receiving e-mail, SMS, proprietaries interfaces, standard data sources (DB rare, the system of industrial automation - sensors). When fully developed system of plugins Voicelant can not even use the phone part of their work, and deal only with the processing and dispatch of data.
