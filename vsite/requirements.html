 Requirements

Voicelant runs under Windows, using boards of Dialogic.

 Equipment
The computer (server) Voicelant
Minimum requirements: Pentium 600 MHz, 128 MB RAM, 40MB Hard Disk space, CD_ROM
Dialogic card
Analog Boards:
  • D4/PCI, D4/PCIU - 4 analogue lines, Scbus no, PCI;
  • D4/PCIUF - 4 analogue lines, Scbus no, PCI, Fax
  • D/4PCIU4S - 4 analogue lines, Scbus no, PCI, Media
  • D/41 ESC PCI - 4 analogue lines, ISA, PCI, Scbus/CTbus
  • D41/JCT-LS - 4 analogue lines, PCI, CTbus, Media, Fax
  • VFX/41JCT-LS - 4 analogue lines, PCI, CTbus, Media, Fax
  • Dialog/4 - 4 analogue lines, Scbus no, ISA;
  • D/120JCT - 12 analogue lines, PCI, Sсbus/Ctbus
  • D/160SC-LS - 16 analogue lines, ISA, Sсbus
  • ProLine/2v - 2 analogue lines, Audio IN/OUT, SCbus no
Digital cards (E1, T1):
  • D/300SC-E1,D/300SC-2E1,D/600SC-2E1 - 1(2)xE1-30(60) channels, 30(60) voice resources, ISA
  • D/240SC-T1,D/240SC-2T1,D/480SC-2T1 - 1(2)xT1-24(48) channels, 24(48) voice resources, ISA
  • D/300PCI-E1 - 1xE1-30 channels, 30 голосовых ресурсов, PCI
  • D/300JCT-E1, D/600JCT-1E1, D/600JCT-2E1 - 1(2)xE1-30(60) channels, 30(60) voice resources, PCI, Media, Fax
  • D/240JCT-T1, D/480JCT-1T1, D/480JCT-2T1 - 1(2)xT1-24(48) channels, 24(48) voice resources, PCI, Media, Fax
  • DM/V600A-2E1, DM/V1200A-4E1 - 2(4)xE1/ISDN PRI-60(120) channels, 60(120) voice resources, PCI, Media
  • DM/V480A-2T1, DM/V960A-4T1 - 2(4)xT1/ISDN PRI-48(96) channels, 48(96) voice resources, PCI, Media
Specifications boards can be found at Dialogic
Dialogic cards can be purchased from distributors of Dialogic and on the web www.acecallcenters.com/, www.tti.net/computer-telephony/dialogic.html , www.ebay.com, www.teleks.com

 Software
Windows
Voicelant supports the following operating systems:
  • Windows NT (Workstation и Server);
  • Windows 2000 (Professional и Server);
  • Windows 2003;
  • Windows XP;
On all operating systems should be installed the latest version of ServicePak's. Also desirable to install Windows Update's
Dialogic Drivers
For Voicelant need to install software Dialogic:
  • Dialogic's System Release 5.1.1+SP1 (Windows NT/2000)(free);
  • Dialogic's System Release 6.0 (Windows XP/2003);
In order to support additional opportunities for Dialogic boards on SR5.1.1 can be installed FeaturePack or ServiceUpdates for SR 6.0. Link to the manufacturer's site: Dialogic
